# mite-assistant-mcp

Java-based MCP server for Mite time tracking.

## Purpose

This project is intended to let AI assistants interact with the Mite API to:

- retrieve time entries
- group time entries for simple reports
- create new time entries

## Planned MCP Tools

For the initial version, the server will focus on these tools:

- `get_time_entries`: Retrieve time entries from Mite.
- `group_time_entries`: Group time entries into simple report-friendly summaries.
- `create_time_entry`: Create a new time entry through the Mite API.

## Development Notes

- Implementation language: Java
- Project type: MCP server
- Primary integration: Mite API
- Keep the first iteration small and focused on time entry access and reporting basics.
